# Light-and-Temp-logger
Forked from Example code for Adafruit data logging shield.
Changing to the sensors that we have.

# IMPORTANT DIRECTIONS.
1. You'll need the RTC lib.
   1. Download it from https://github.com/adafruit/RTClib
   2. Extract it into the libraries folder
   3. MAKE SURE THE NAME OF THE FOLDER IS "RTClib" (no quotes, no -master, etc)
   4. Restart Arduino IDE


Eventually adding some serial interaction with user.
